# Audit microcopy & voix — Coach Objectifs

_Réalisé le 2026-06-21 par un sous-agent IA UX writer SaaS B2B._

## 1. Diagnostic du ton actuel

- Ton descriptif-encyclopédique : décrit ce que fait l'outil, pas ce que l'utilisateur va vivre.
- Verbiage « tutoriel » lourd : structure interrogative-polie, posture de prof.
- Aucune voix, aucun rythme : phrases plates, pas d'ellipse, pas de jeu sur la ponctuation.
- Statuts cliniques sans énergie : juste, mais désincarné.

## 2. Voice & Tone cible — 5 principes

1. **Complice expert, pas prof.** « Choisis un seul type à la fois » → « Un type à la fois — on garde la pédagogie cohérente. »
2. **Concision Stripe / Linear.** Une phrase = une idée.
3. **Verbe d'action en tête, pas de question rhétorique.** « Sur quel type d'objectif veux-tu travailler ? » → « Choisis ton terrain. »
4. **Clin d'œil sec, jamais gimmick.** Un trait d'esprit par écran max. Pas d'emoji, pas de « bravo ».
5. **Pédagogie incarnée, pas neutre.** « À reformuler » → « À refaire — on n'y est pas. »

## 4. Vocabulaire

**À bannir** : « Veux-tu… ? », « Tu peux… », « N'hésite pas », « Cliquez sur le bouton ci-dessous », « Bravo / Félicitations / Champion », « Unlock / Empower / Débloquer ton potentiel », « Une erreur est survenue », parenthèses lourdes.

**À favoriser** : « On y va », « On regarde », « On reprend » (le « on » complice atelier). « À toi », « À ton tour ». Phrases nominales : « Réécrit. », « Solide. », « Presque. ». Adverbes signature en fin : « Toujours. », « Pas plus. ». Verbes nus impératifs : « Écris. », « Glisse. », « Vise. ». Vocabulaire effort physique : « muscler », « échauffement », « cruise », « corsé », « tenir ». Anglicismes tech assumés courts : « live », « solid ».
